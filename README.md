#Portal 

##假设场景一
	创业公司的老板，事业板块发展的不错，slack、trello、tower、钉钉、云之家等产品都再用，或者很了解，
	但还是有一些需求需要定制，才能满足自身业务发展。
	传统方式会找供应商，帮忙完成咨询和实施。
现在我们要挑战新的方式，实现：业务领域下的快速业务创新、实施平台。
